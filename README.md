# 地图标记 Flutter 模块（rider_map_marker_flutter）

Flutter 地图标记相关模块（rider_map_marker_flutter），封装地图标记 / 标注的跨平台实现，可作为骑行 / 配送类应用的前端地图能力底座。

## 功能特性

- 地图标记 / 标注封装
- 跨平台（Android / iOS / Web）Flutter 实现
- 可嵌入宿主应用复用
- 组件化接口设计

## 技术栈

Flutter / Dart；地图相关插件依赖；模块化 package 结构。
